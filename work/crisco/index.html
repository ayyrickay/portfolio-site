--- layout: page title: "Crisco" deliverable: 'url shortener website' role: "front-end developer" problem: "how the heck does a url shortener work?" methods: "web development: flask, sqlite, bootstrap" artifacts: 'technical report' solution: "Crisco focused more on technical aspects than aesthetics. My classmate and I started out developing the framework for a URL shortener using python's flask library, and then added on a number of technical features to improve the UX. This included a random URL generator (rather than requiring a tag), a dupe-free tag database that generated a unique 'random' URL for a given link, and mobile responsiveness. We also tried to keep the project lighthearted, and went with the southern theme and a fun 404 page. This project functioned as an exercise in process and technique - so we figured we might as well keep it lighthearted!" ---
